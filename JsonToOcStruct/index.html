JSON to Origin C Struct

Convert a JSON object into an Origin C struct.

This tool converts a JSON object into the struct(s) required to process the JSON data in Origin C using the Origin C functionJSON.ReadString. Paste your JSON object on the left and the struct(s) will be output to the right for you to copy to your Origin C source code. Checking Verbose Mode will include helpful comments in the output.

The main struct (the one to pass to JSON.ReadString) is always declared last.

Please read: For an explanation of why "Child" is used in struct names, limitations of determining data types, and potential issues with struct member naming see this documentation.

For an example, try converting JSON from the PUG REST API.
